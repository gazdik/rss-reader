# Sample RSS Feeds for Testing

Here are some popular RSS feeds you can use to test the application:

## News & Technology

- **Hacker News**: https://news.ycombinator.com/rss
- **TechCrunch**: https://techcrunch.com/feed/
- **The Verge**: https://www.theverge.com/rss/index.xml
- **Ars Technica**: https://feeds.arstechnica.com/arstechnica/index
- **Wired**: https://www.wired.com/feed/rss

## Development & Programming

- **CSS-Tricks**: https://css-tricks.com/feed/
- **Smashing Magazine**: https://www.smashingmagazine.com/feed/
- **Dev.to**: https://dev.to/feed
- **A List Apart**: https://alistapart.com/main/feed/

## Blogs

- **Joel on Software**: https://www.joelonsoftware.com/feed/
- **Paul Graham**: http://www.aaronsw.com/2002/feeds/pgessays.rss

## Reddit

- **r/programming**: https://www.reddit.com/r/programming/.rss
- **r/webdev**: https://www.reddit.com/r/webdev/.rss
- **r/technology**: https://www.reddit.com/r/technology/.rss

## GitHub

- **GitHub Blog**: https://github.blog/feed/

## Note

Some feeds may require CORS proxy to work. The application automatically uses a CORS proxy (allorigins.win) to fetch feeds.
